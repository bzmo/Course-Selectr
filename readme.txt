To use my application, select a subject from the list, type in any prefix string
of a course number, and select a term. Then, click on "Submit".

Your results will be displayed below. If there are courses listed, click on
any individual entry to show its course info. Click on the same individual entry
to hide its course info.

Example of a query: CS, 349, N/A => CS349: User Interfaces shows up.
Example of a query: CS, 34, N/A => CS341, CS343, CS348, CS349 all show up.
